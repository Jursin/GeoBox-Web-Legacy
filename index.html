使用技巧

  • 将编辑好的 GeoGebra 文件调整好窗口大小再保存，打包编译后将尽量保持窗口宽高比自适应不同分辨率，建议使用 16:9 的宽高比例
  • 尽量将体积小的 GeoGebra 文件拖放到最前面，以提高打包文件的打开速度
    • 打包编译：
      • EXE可执行文件(推荐)：生成独立运行的单一可执行文件，无需安装任何 GeoGebra 环境，但有被防火墙和杀毒软件误报的风险
      • 在线网页文件：生成单一网页文件，可在浏览器中打开（需联网），适合在线分享使用
      • 离线网页文件：生成单一网页文件，需下载2.0版本离线包（注：将离线包解压缩，再将导出的 HTML 文件放到解压缩后的 geobox2_0_offline 文件夹内打开）
  • 播放打包文件时可点击工具栏中的页码跳转到指定页面，点击 橡皮 按钮可以进入擦除状态
  • 播放时还支持激光电子笔和键盘 PgUp 和 PgDn 键前后翻页
  • Windows 7 和部分前期 Windows 10 系统需要安装 WebView2 Runtime 才能正常使用
